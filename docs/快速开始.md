# 快速开始

## 环境要求

- Node.js 18+
- Cesium ^1.120.0（作为 peerDependency，需自行安装）

## 安装

```bash
npm install wiselayer cesium
```

## 最小 WMS 示例

```typescript
import { Viewer } from 'cesium';
import { createLayer } from 'wiselayer';
import 'cesium/Build/Cesium/Widgets/widgets.css';

const viewer = new Viewer('cesiumContainer');

const wmsLayer = createLayer({
  type: 'wms',
  url: 'https://ahocevar.com/geoserver/wms',
  layers: 'topp:states',
  crs: 'EPSG:4326',
});

wmsLayer.addTo(viewer);
```

## 最小 WMTS 示例

```typescript
import { createLayer } from 'wiselayer';

const wmtsLayer = createLayer({
  type: 'wmts',
  url: 'https://services.arcgisonline.com/arcgis/rest/services/World_Imagery/MapServer/WMTS',
  layer: 'World_Imagery',
  style: 'default',
  format: 'image/jpeg',
  tileMatrixSetID: 'default028mm',
});

wmtsLayer.addTo(viewer);
```

## 最小天地图示例

Token 需在 [天地图开发者控制台](https://console.tianditu.gov.cn/) 申请。

```typescript
import { createLayer } from 'wiselayer';

const token = 'YOUR_TOKEN';

// 矢量底图 + 注记（Web 墨卡托，默认 crs: EPSG:3857）
createLayer({ type: 'tianditu', token, imageType: 'vector' }).addTo(viewer);
createLayer({ type: 'tianditu', token, imageType: 'vectorAnnotation', zIndex: 1 }).addTo(viewer);
```

原生 HTML 示例见 `examples/native/01_layer_tianditu.html`。

## 与 Vite 集成

推荐使用 `vite-plugin-cesium` 处理 Cesium 静态资源：

```bash
npm install -D vite-plugin-cesium
```

```typescript
// vite.config.ts
import { defineConfig } from 'vite';
import cesium from 'vite-plugin-cesium';

export default defineConfig({
  plugins: [cesium()],
});
```

完整可运行示例见仓库 `examples/vite-cesium/` 目录。

## 典型工作流

```typescript
import { Viewer, Rectangle } from 'cesium';
import { createLayer, addLayers } from 'wiselayer';

const viewer = new Viewer('cesiumContainer');
viewer.imageryLayers.removeAll(); // 可选：清除默认底图

// 方式一：单个创建
const base = createLayer({
  type: 'wmts',
  url: '...',
  layer: 'World_Imagery',
  tileMatrixSetID: 'default028mm',
  zIndex: 0,
});
base.addTo(viewer);

// 方式二：批量添加
const layers = addLayers(viewer, [
  { type: 'wms', url: '...', layers: 'states', zIndex: 1, rectangle: Rectangle.fromDegrees(-125, 24, -66, 50) },
]);

// 图层操作
layers[0].setAlpha(0.8);
layers[0].setVisible(true);
layers[0].centerAt({ duration: 1.5 });
layers[0].setZIndex(0);
```
